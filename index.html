Desktop Application for Android Screen Mirroring

Zone Cast

Zone Cast is a desktop application that simplifies Android screen mirroring by providing a modern graphical interface for the scrcpy engine. Instead of using command-line commands, users can configure video quality, frame rate, and connection settings through an intuitive interface before starting a mirroring session.

The application was built with a focus on usability, allowing users to connect their Android devices quickly while monitoring connection status and streaming performance.

Key Features

  • Modern desktop user interface
  • Android screen mirroring
  • Adjustable video bitrate
  • Configurable frame rate (FPS)
  • Device status monitoring
  • Live event logging
  • Simple one-click connection

Technologies Used

HTML5 CSS3 JavaScript Node.js ADB scrcpy

Custom Android Screen Sharing & Remote Control System

Infinity Cast

Infinity Cast is a complete screen sharing solution developed from scratch without relying on third-party mirroring tools. The project consists of two applications working together to enable seamless screen streaming and remote control.

📱 Android Application (Android Studio)

Responsible for capturing the device screen and transmitting the video stream over TCP/IP.

💻 Windows Desktop Application (Qt Creator)

Responsible for receiving the stream, displaying it in real time, and enabling remote control of the Android device.

Key Features

  • Custom screen streaming implementation
  • Android-to-Windows screen sharing
  • TCP/IP socket communication
  • Low-latency streaming
  • Remote device control
  • Cross-platform architecture

Technologies Used

Android Application

Java Android Studio MediaProjection API TCP/IP Networking

Desktop Application

C++ Qt Framework Qt Creator TCP/IP Sockets Multithreading
